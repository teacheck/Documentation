Aurkezpena
==========

Erabilera
---------

``main.tex`` fitxategiaren hasieran hiru aldagai daude proiektuaren
azalekoa eta sortzen den pdf fitxategiaren metainformazioa konfiguratzeko:

- "\egilea"     : egilearen izena
- "\izenburua"  : proiektuaren izenburua da}
- "\data"       : memoriaren azalekoan agertzen den data

- "\euskaraz"   : lerro honek adierazten du memoria euskaraz ala gazteleraz
  egiten den. Komentatu gazteleraz egiteko.

.. note:: 

	Fitxategien kodeketak utf-8 izan behar du. Honek ahalbideratzen du
	tildeak eta "ñ" zuzenean erabili ahal izatea.

.. warning::

	Tildeak erabili badaitezke ere (utf-8 erabiltzeari esker),
	dokumentuaren aurkibidean agertuko diren izeburuetan (chapter, section,
	subsection) tildeak ez jartzea gomendatzen da:

	- á => \'{a}
